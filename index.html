Hi everyone! I'm James!
A web developer

My name is James Pham, and I am an aspiring web developer! I started my coding journey mid-2017 when I decided to make a career change. Khan Academy, Code Academy, and EDX gave me with my very first glimpse of computer programming. A few months later, I enrolled at my local community college to further enhance my coding education. It wasn't until after my first year at De Anza College when I realized that there was a whole other world of coding dedicated to the web development, and how much fun it was and still is! Ever since my discovery of web development, my fascination for web applications has driven me to research, learn, and practice web development.

My skillset primarily focuses on front-end web development. The tools in my web applications include: HTML/CSS, JavaScript, jQuery, Bootstrap, Materialize, D3, and React. I have many projects that represent my knowledge of the following web developer technologies.

Although my work concentrates on front-end programming and web design, I also have knowledge of back-end tools such as: Node, Express, MongoDB, Mongoose, AJAX, and many others. I love knowing how things "work under the hood" and because front-end and back-end work is so intertwined, it only made sense for me to pick up these additional skills.
